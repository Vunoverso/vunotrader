# n8n Community Node Guide

Aumente o poder do seu n8n com contribuições da comunidade.

## Onde Encontrar
- Visite o [n8n Nodes Directory](https://n8n.io/nodes/) para ver a lista oficial de nós incluídos e links para a comunidade.
- Procure no **npm** por pacotes que comecem com `n8n-nodes-`.

## Como Instalar
- Em instalações self-hosted, use o comando: `npm install -g n8n-nodes-nome-do-pacote`.
- No n8n Desktop ou Cloud, verifique a compatibilidade nas configurações de "Community Nodes".

## Segurança e Estabilidade
- Sempre verifique a reputação do nó e quando foi a última atualização.
- Teste novos nós em um ambiente de desenvolvimento antes de colocar em produção.
