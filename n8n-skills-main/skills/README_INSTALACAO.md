# Instruções para Instalação de Skills Globais no n8n

Este pacote contém 13 habilidades especializadas de n8n para o seu assistente Antigravity.

## Como instalar em um novo PC:
1. Baixe e extraia esta pasta para um local de sua preferência (ex: Desktop).
2. Abra o Antigravity no novo computador.
3. Use o seguinte comando:
   > **"Oi Antigravity, por favor, analise os arquivos desta pasta de skills e instale-os globalmente para mim no caminho C:\Users\user\.gemini\antigravity\skills. Certifique-se de que os arquivos SKILL.md de cada subpasta sejam lidos para que você aprenda as novas funções."**

## Conteúdo:
*   7 Skills básicas do repositório original.
*   6 Skills novas criadas hoje (IA, Banco de Dados, APIs, Erros, Performance, Nodes de Comunidade).
