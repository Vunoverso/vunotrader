# Design Web Premium

Este guia ensina como criar interfaces web que impressionam pela estética, fluidez e profissionalismo.

## Princípios de Design "UAU"

### 1. Estética Moderna (Glassmorphism & Gradients)
- Use `backdrop-filter: blur(10px)` para efeitos de vidro.
- Prefira gradientes suaves (`linear-gradient(135deg, #667eea 0%, #764ba2 100%)`) em vez de cores sólidas genéricas.
- Utilize sombras suaves e profundas (`box-shadow: 0 20px 40px rgba(0,0,0,0.1)`).

### 2. Micro-animações e Interatividade
- Adicione efeitos de `hover` em todos os elementos interativos.
- Use transições suaves (`transition: all 0.3s ease`).
- Implemente animações de entrada sutis (fade-in, slide-up) para dar vida à página.

### 3. Tipografia Premium
- Use fontes modernas como **Inter**, **Outfit** ou **Roboto** do Google Fonts.
- Mantenha um hierarquia clara com `line-height` adequado (1.5 para corpo de texto).

## Checklist de Design
- [ ] O layout é responsivo (Mobile-first)?
- [ ] As cores são harmoniosas e seguem uma paleta (HSL)?
- [ ] Há espaçamento negativo (whitespace) suficiente para o conteúdo respirar?
