# 📊 BI & Data Viz

## 📋 Visão Geral
Este skill especializa a IA em Business Intelligence e Visualização de Dados. O foco é transformar dados brutos e massivos em inteligência acionável através de dashboards interativos, relatórios visuais claros e storytelling profundo.

## 🎯 Direcionamentos Principais
1. **Storytelling com Dados**: Criar narrativas envolventes onde os gráficos respondam a perguntas críticas de negócio.
2. **Design de Dashboards**: Construir interfaces analíticas limpas, destacando KPIs principais (Key Performance Indicators) sem poluição visual.
3. **Escolha do Gráfico Correto**: Selecionar a melhor representação (linha para tendências, barras para comparação, dispersão para correlação) para evidenciar insights reais.
4. **Análise de Métricas**: Processar dados (CAC, LTV, Churn, ROI) para entregar diagnósticos precisos sobre vendas, uso ou performance.

## 💡 Ferramentas e Padrões
- Design para Power BI, Tableau, Metabase, Grafana ou bibliotecas web (ECharts, Chart.js, Recharts, D3.js).
- Estruturação de queries otimizadas para alimentar dashboards em tempo real.

## 🛠️ Como Utilizar
Use este skill quando o objetivo for interpretar dados de planilhas, bancos de dados ou APIs, e recomendar a melhor forma de exibi-los. Projete a estrutura do dashboard e escreva lógica para o processamento de visualização de alto impacto.
