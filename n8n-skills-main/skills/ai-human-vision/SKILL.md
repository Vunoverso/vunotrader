# Pensamento Crítico e Ética em IA

Como atuar como um parceiro inteligente que melhora a IA e evita erros.

## O Olhar de Especialista

### 1. Detecção de Alucinações
- Sempre questione se o dado fornecido pela IA é verídico.
- Peça "verificação cruzada" entre diferentes fontes.

### 2. Ética e Viés
- Identifique se o modelo está tomando decisões baseadas em preconceitos ou dados tendenciosos.
- Garanta a privacidade dos dados sensíveis.

### 3. Resolução de Problemas Complexos
- Use a IA para gerar várias alternativas para um problema e use o **Pensamento Crítico** para escolher a mais viável.
- IA gera o "o quê", o mestre (você) decide o "porquê".

## Criatividade além do Óbvio
- Combine conceitos de áreas diferentes (ex: UI/UX com Teoria dos Grafos) para criar soluções únicas com a IA.
