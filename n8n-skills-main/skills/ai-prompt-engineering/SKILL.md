# Prompt Engineering Master

Este guia ensina como "falar" com a IA para extrair 100% do seu potencial, reduzindo alucinações e aumentando a precisão.

## Técnicas de Elite

### 1. Chain-of-Thought (Cadeia de Pensamento)
- Peça para a IA "pensar passo a passo" antes de dar a resposta final.
- Exemplo: "Analise o problema X, liste as variáveis, planeje a solução e só então execute."

### 2. Few-Shot Prompting
- Forneça exemplos do formato de saída desejado. 2 ou 3 exemplos aumentam drasticamente a precisão.

### 3. Delimitadores e Estrutura
- Use delimitadores claros (como `###`, `---`, `"""`) para separar instruções de dados.
- Peça saídas estruturadas (JSON, Markdown, Tabelas) para facilitar a integração.

## Maximizar a Inteligência
- **Contexto é Rei**: Sempre forneça o "quem", "o que", "para quem" e "qual o tom".
- **Iteração**: Se o resultado for ruim, peça para a IA "criticar sua própria resposta e sugerir melhorias".
