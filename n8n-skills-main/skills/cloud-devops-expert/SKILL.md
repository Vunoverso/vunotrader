# 🏗️ Cloud & DevOps

## 📋 Visão Geral
Este skill foca em Infraestrutura como Código (IaC), DevOps e computação em nuvem (AWS, GCP, Azure, Vercel). Capacita a IA a planejar orquestrações complexas, pipelines de CI/CD automatizados e ambientes de alta disponibilidade, tolerantes a falhas.

## 🎯 Direcionamentos Principais
1. **Containerização & Orquestração**: Desenho e configuração eficiente de Dockerfiles, Docker Compose e manifestos do Kubernetes (K8s) visando performance.
2. **Pipelines de CI/CD**: Automação de testes, builds e deploys (GitHub Actions, GitLab CI, Jenkins), promovendo integração contínua sem quebras de ambiente.
3. **Escalabilidade & Alta Disponibilidade**: Arquitetar microserviços, Load Balancers, Auto-scaling groups e Content Delivery Networks (CDNs) resilientes.
4. **Monitoramento e Observabilidade**: Implementar logs centralizados, tracing distribuído e métricas (Prometheus, Grafana, ELK Stack, Datadog).

## 💡 Aplicações Práticas
- Criação de scripts Terraform para provisionamento de infra cloud.
- Resolução de gargalos de performance em servidores Linux (Ubuntu, Alpine).
- Elaboração de infraestrutura segura de rede (VPCs, Subnets, Security Groups).

## 🛠️ Como Utilizar
Acione ao precisar estruturar servidores, resolver bugs de deployment, automatizar builds ou migrar aplicações monolíticas para arquiteturas cloud-native e serverless.
