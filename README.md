# Vuno Trader\n\nPlataforma SaaS de robo trader com execucao no MT5, cerebro externo em Python, backend FastAPI, frontend Next.js e persistencia no Supabase.\n\n## Estrutura\n\n- `web/`: app web (Next.js)\n- `backend/`: API e workers (FastAPI)\n- `supabase/`: migrations e estrutura de banco\n- `scripts/`: automacoes locais (CDP, setup, utilitarios)\n- `projeto/`: documentos de arquitetura e planejamento\n- `evolucao/`: historico tecnico das decisoes e mudancas\n- `vunotrader_brain.py`: servico de decisao do robo\n\n## Requisitos\n\n- Python 3.11+\n- Node.js 20+\n- Conta Supabase\n\n## Setup rapido\n\n### Backend\n\n1. Entrar em `backend/`\n2. Criar e ativar ambiente virtual\n3. Instalar dependencias de `backend/requirements.txt`\n4. Configurar `backend/.env`\n5. Rodar API\n\n### Frontend\n\n1. Entrar em `web/`\n2. Instalar dependencias\n3. Rodar app de desenvolvimento\n\n### Brain Python\n\n1. Configurar `brain.env` a partir de `brain.env.example`\n2. Instalar dependencias de `brain-requirements.txt`\n3. Executar `vunotrader_brain.py`\n\n## Publicacao no GitHub\n\nEste repositorio ja inclui regras de ignore para evitar envio de credenciais locais (`*.env`) e artefatos de build/venv.\n
